筛选选择器

所有的筛选选择器都是API方法

**find(selector)**表示在后代元素中查找

**children(selector)**表示在子代元素中查找

**parent()**表示父亲层级

**eq(index)**中传入jquery伪数组中的索引值

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
    <div class="wrap">
        <div id="box">
            <p id="zero">这是一个段落</p>
        </div>
        <ul>
            <li>
                <p>这是第一个段落</p>
            </li>
            <li>
                <p id="two">这是第二个段落</p>
            </li>
            <li>
                <p>这是第三个段落</p>
            </li>
        </ul>
    </div>

    <script type="text/javascript" src="./js/jquery-3.3.1.js"></script>
    <script type="text/javascript">
        // jquery对象下的API方法，find(selector)，支持所有的选择器，查找后代的子子孙孙
        // 按标签选择器查找
        $('.wrap').find('p').css('color','red');

        // 按id查找
        $('.wrap').find('#two').css('font-size',30);

        // 只查找亲儿子层级的标签，children(selector)，支持所有的选择器
        $('#box').children('p').css('color','green');
        
        // 链式变成，$('#box').children('p').css('color','green')返回的是一个jquery对象，仍然可以调用API，执行click事件
        $('#box').children('p').css('color','green').click(function(){
            alert($(this).text());
        });

        // 只查找父亲层级的对象
        $('#zero').parent().css({
            'font-size':25,
            'width':'200px',
            'height':'200px',
            'background':'blue'
        });

        // 按索引查找，此eq不是过滤选择器下的jqueryobj:eq(2)，而是一个eq方法，如jquery.eq(1)，同样也是equal
        $('.wrap').find('p').eq(1).css({
            'font-size':30,
            'color':'blue'
        })
    </script>
</body>
</html>
```
