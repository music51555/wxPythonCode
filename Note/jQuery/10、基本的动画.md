基本的动画

show()，显示元素

hide()，隐藏属性

toggle()，显示、隐藏开关

实则修改的是display

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style type="text/css">
        div{
            width: 100px;
            height: 100px;
            background-color: red;
            display: none;
        }
        button{
            width: 100px;
            height: 50px;
        }
    </style>
</head>
<body>
    <button id="show">显示</button>
    <button id="hide">隐藏</button>
    <button id="on_off">开关</button>
    <div class="box"></div>
    <script type="text/javascript" src="./js/jquery-3.3.1.js"></script>
    <script type="text/javascript">
    // 显示或隐藏动画
        $('#show').click(function(){
            // 显示元素，格式(时间，函数)
            $('.box').show(500,function(){
                $(this).text('显示出来了');
            });
        });

        $('#hide').click(function(){
            // 隐藏元素
            $('.box').hide(2000);
        });
    
    // 美 /'tɑɡl/ 开关，触发器，开关式动画，一个按钮控制动画的显示和隐藏，在开关动画前，先停止前一个动画的动作
    $('#on_off').click(function(){
        $('.box').stop().toggle(2000);
    });
    
    </script>
</body>
</html>
```



默认normal 400ms，slow 600ms，fast 200ms
