**表的数据类型：**

1、整型

**tinyint**

```python
占用1bytes==8bit

有符号取值范围：-128 ~ 127

#设置有符号的tinyint数据类型，数据类型位宽默认显示为4位，也就是取值范围的位宽，符号+数据，-120
alter table t0 add age tinyint;
+-------+------------+------+-----+---------+-------+
| Field | Type       | Null | Key | Default | Extra |
+-------+------------+------+-----+---------+-------+
| id    | tinyint(4) | YES  |     | NULL    |       |
+-------+------------+------+-----+---------+-------+


#当设置超出范围的数据时，会报错
mysql> insert into t0 set age=-129;
ERROR 1264 (22003): Out of range value for column 'age' at row 1
mysql> insert into t0 set age=129;
ERROR 1264 (22003): Out of range value for column 'age' at row 1


无符号取值范围：0 ~ 255
#设置无符号的tinyint数据类型，表结构中有unsigned标识，数据类型位宽默认显示为3位，也就是取值范围的位宽
create table t7(id tinyint unsigned);
+-------+---------------------+------+-----+---------+-------+
| Field | Type                | Null | Key | Default | Extra |
+-------+---------------------+------+-----+---------+-------+
| id    | tinyint(3) unsigned | YES  |     | NULL    |       |
+-------+---------------------+------+-----+---------+-------+

#当设置超出范围的数据时，会报错
mysql> insert into t7 set id=256;
ERROR 1264 (22003): Out of range value for column 'id' at row 1
mysql> insert into t7 set id=-1;
ERROR 1264 (22003): Out of range value for column 'id' at row 1
```

**bigint**

```python
占用8bytes，即32个bit，即2**32

有符号取值范围：-9223372036854775808 ～ 9223372036854775807
#设置有符号为的bigint类型
create table t10(id bigint);
+-------+------------+------+-----+---------+-------+
| Field | Type       | Null | Key | Default | Extra |
+-------+------------+------+-----+---------+-------+
| id    | bigint(20) | YES  |     | NULL    |       |
+-------+------------+------+-----+---------+-------+


无符号取值范围：0 ～ 18446744073709551615
#设置有符号为的bigint类型
create table t10(id bigint unsigned);
+-------+---------------------+------+-----+---------+-------+
| Field | Type                | Null | Key | Default | Extra |
+-------+---------------------+------+-----+---------+-------+
| id    | bigint(20) unsigned | YES  |     | NULL    |       |
+-------+---------------------+------+-----+---------+-------+
```



**填充位宽：**

```python
当设置整型的位宽时，不是对其设置占用几个字节，整形数据类型默认使用的是数据类型的长度
如果设置位宽，是设置的显示位宽，而且设置的意义不大，也就是当前字符在表格中显示的宽度，可以设置zerofill参数来用0填充位宽
#设置整形位宽，注意没有逗号分隔
create table t12(id int(5) zerofill);

#插入数据后显示状态
+-------+
| id    |
+-------+
| 00003 |
+-------+

```



**2、浮点型**

存储薪资、身高、体重等

**float**

```python
#指定了最大位宽255位，其中小数点可以占用30位，float类型和double类型对比于decimal类型，虽然精度没有那么高，但是整数位的位宽多，在日常使用情况下float类型就足够了
create table t1(height float(255,30));

mysql> desc t1;
+--------+---------------+------+-----+---------+-------+
| Field  | Type          | Null | Key | Default | Extra |
+--------+---------------+------+-----+---------+-------+
| height | float(255,30) | YES  |     | NULL    |       |
+--------+---------------+------+-----+---------+-------+

insert into t1 values(1.11111111111111111111);
#非准确小数值，会随着小数的增多，精度变得不准确
mysql> select * from t1;
+----------------------------------+
| height                           |
+----------------------------------+
| 1.111111164093017600000000000000 |
+----------------------------------+
```

**decimal**

