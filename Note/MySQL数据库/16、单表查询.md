**sql语句执行优先级：**

1、先找到表 from student

2、where条件

3、group by分组

4、having过滤

5、执行select

6、去重

7、order by排序

8、limit n限制显示结果的条数



```mysql
#简单查询
select id,name from student;
select * from student;

#distinct关键去重
mysql> select distinct post from employee;
+-----------------------------------------+
| post                                    |
+-----------------------------------------+
| 老男孩驻沙河办事处外交大使                  |
| teacher                                 |
| sale                                    |
| operation                               |
+-----------------------------------------+

#四则运算，查询出的结果是一张虚拟表
mysql> select name,salary*12 from employee;
+------------+-------------+
| name       | salary*12   |
+------------+-------------+
| egon       |    87603.96 |
| alex       | 12000003.72 |
| wupeiqi    |    99600.00 |
| yuanhao    |    42000.00 |
| liwenzhou  |    25200.00 |


#为查询结果列设置别名,其中as可以不写
mysql> select name,salary as annual_salary from employee;

| name       | annual_salary |
+------------+---------------+
| egon       |       7300.33 |
| alex       |    1000000.31 |

#通过concat关键字拼接字符串，把姓名和性别合并为一列，根本原理使利用逗号分隔字符串
mysql> select concat('姓名：',name,',','性别：',sex) as info,salary from employee;

| info                                | salary     |

| 姓名：egon,性别：male                 |    7300.33 |
```
