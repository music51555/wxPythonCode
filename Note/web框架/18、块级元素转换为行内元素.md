块级元素转换为行内元素

每个块级元素都是另起一行显示的，但是添加**display:inline**表示将块状元素转换为行内元素，变为行内元素后，width、height参数失效，因为行内元素不支持，且box1和box2的div标签会在一行中显示

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style type="text/css">
        /* 添加display:inline表示将块状元素转换为行内元素，变为行内元素后，width、height参数失效，因为行内元素不支持，且box1和box2的div标签会在一行中显示 */
        .box1{
            display: inline;
            width: 300px;
            height: 30px;
            font-size: 18px;
            border: 1px solid red;
        }
        .box2{
            display: inline;
            margin-top: 20px;
            width: 300px;
            height: 30px;
            border: 1px solid green;
        }
    </style>
</head>
<body>
    <div class="box1">
        第一个盒子
    </div>
    <div class="box2">
        第二个盒子
    </div>
</body>
</html>
```



添加**display: block**后，行内元素将转换为块状元素，每一行会独占一行显示，也可以设置宽高属性

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style type="text/css">
    /* 添加display: block后，行内元素将转换为块状元素，每一行会独占一行显示，也可以设置宽高属性 */
        span{
            color:red;
            display: block;
            width: 200px;
            height: 50px;
            border: 1px solid red;
        }
    </style>
</head>
<body>
    <span>这是一小点儿内容</span>
    <span>这是一小点儿内容</span>
</body>
</html>
```



添加**display: inline-block**后，行内元素将转换为行内块元素，在一行中可以显示多个行内快元素，也可以设置宽高属性

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style type="text/css">
    /* 添加display: inline-block后，行内元素将转换为行内块元素，在一行中可以显示多个行内快元素，也可以设置宽高属性 */
        span{
            color:red;
            display: inline-block;
            width: 200px;
            height: 50px;
            border: 1px solid red;
        }
    </style>
</head>
<body>
    <span>这是一小点儿内容</span>
    <span>这是一小点儿内容</span>
</body>
</html>
```



当标签样式设置**display: none**后，在网页中将被隐藏掉

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style type="text/css">
        /* 当标签样式设置display: none后，在网页中将被隐藏掉 */
        #span1{
            color:red;
            display: none;
            width: 200px;
            height: 50px;
            border: 1px solid red;
        }
        #span2{
            color:red;
            display: block;
            width: 200px;
            height: 50px;
            border: 1px solid red;
        }
    </style>
</head>
<body>
    <div>
        这是一块内容
    </div>
    <span id="span1">这是一小点儿内容11</span>
    <span id="span2">这是一小点儿内容22</span>
</body>
</html>
```



当标签样式设置**visibility: hidden 美 /'vɪzə'bɪləti/  **能见度，可见性，在网页中将被隐藏掉，但是所在区域会被留下来

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style type="text/css">
        /* 当标签样式设置visibility: hidden后，在网页中将被隐藏掉，但是所在区域会被留下来 */
        #span1{
            color:red;
            visibility: hidden;
            width: 200px;
            height: 50px;
            border: 1px solid red;
        }
        #span2{
            color:red;
            display: block;
            width: 200px;
            height: 50px;
            border: 1px solid red;
        }
    </style>
</head>
<body>
    <div>
        这是一块内容
    </div>
    <span id="span1">这是一小点儿内容11</span>
    <span id="span2">这是一小点儿内容22</span>
</body>
</html>
```
