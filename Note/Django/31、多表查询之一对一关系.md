多表查询之一对一关系

如作者表和个人信息表

一个作者对应一条个人信息

一条个人信息对应一个作者

所以是一对一的关系，在哪张表中建立关联字段都是可以的，但**必须是unique唯一的**，如在作者表中

```
id    name    auth_id(但是对应的个人信息id必须是unique唯一的)
1     alex      1
2     egon      2
```



个人信息表

```
id    addr    sex     phone
 1    昌平     male    18311902221
 2    平谷     female  13099188002
```
