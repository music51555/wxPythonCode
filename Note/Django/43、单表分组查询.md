单表分组查询

查询每一个部门的名称和平均薪资

```python
def query():
    # 单表模型.objects.values(分组的依据).annotate(聚合函数)，annotate表示统计什么，
    # Book.objects.all()表示select * from book,Book.objects.values('id')表示select id from book,所以在单表的情况下对id和对all()进行group by没有意义，因为包含主键的数据都是唯一的，没有意义
    # 在以什么为分组的依据时，也会打印该列值
    # 返回值是<QuerySet [{'dep': '市场部', 'salary__avg': 1000.0}, {'dep': '研发部', 'salary__avg': 1500.0}, {'dep': '运营部', 'salary__avg': 1200.0}]>
    ret=Emp.objects.values('dep').annotate(Avg('salary'))
    print(ret)

    return HttpResponse('OK')
```
