模板语法之继承

#### 一、include继承

##### {% include 'panel.html' %}引入一段代码

##### 1、首先在`template`文件夹下创建`panel.html`文件，存储的是bootstrap的面板代码

```html
<div class="panel panel-primary">
    <div class="panel-heading">Panel heading without title</div>
    <div class="panel-body">
        Panel content
    </div>
</div>
<div class="panel panel-danger">
    <div class="panel-heading">Panel heading without title</div>
    <div class="panel-body">
        Panel content
    </div>
</div>
<div class="panel panel-warning">
    <div class="panel-heading">Panel heading without title</div>
    <div class="panel-body">
        Panel content
    </div>
</div>
```



##### 2、在`order.html`代码中，使用`{include 'order.html'}`来引入对应文件的html代码，这样在展示文件时就可以正确加载其他HTML文件中的内容

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>order</title>
    <link rel="stylesheet" href="/static/bootstrap-3.3.7/css/bootstrap.css">
    <style type="text/css">
        * {
            margin: 0;
            padding: 0;
        }
        .topbar {
            width: 100%;
            height: 50px;
            background-color: #369;
        }
    </style>
</head>
<body>
<div class="topbar"></div>
<div class="container">
    <div class="row">
        <div class="col-md-4">
            <div class="thumbnail">
                <!--使用include标签，插入指定panel.html文件中的全部内容到thumbnail盒子中-->
                {% include 'panel.html' %}
            </div>
        </div>
        <div class="col-md-8">
            <div class="thumbnail">
                order
            </div>
        </div>
    </div>
</div>
</body>
</html>
```



#### 二、extends继承

##### {% extends 'panel.html' %}继承一段代码

与`include`的区别是，`include`是加载`html`代码块儿，而`extends`是继承文件中的所有`html`代码，并可以使用`block`盒子，在`panel.html`文件中包含面板代码，和预留的`block`盒子代码

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <!--在html网页中可以预留多个block盒子，包含父盒子原有内容-->
    {% block title %}
    <title>panel</title>
    {% endblock %}
    <link rel="stylesheet" href="/static/bootstrap-3.3.7/css/bootstrap.css">
    <style type="text/css">
        * {
            margin: 0;
            padding: 0;
        }
        .topbar {
            width: 100%;
            height: 50px;
            background-color: #369;
        }
    </style>
</head>
<body>
<div class="topbar"></div>
<div class="container">
    <div class="row">
        <div class="col-md-4">
            <div class="thumbnail">
                <div class="panel panel-primary">
                    <div class="panel-heading">Panel heading without title</div>
                    <div class="panel-body">
                        Panel content
                    </div>
                </div>
                <div class="panel panel-danger">
                    <div class="panel-heading">Panel heading without title</div>
                    <div class="panel-body">
                        Panel content
                    </div>
                </div>
                <div class="panel panel-warning">
                    <div class="panel-heading">Panel heading without title</div>
                    <div class="panel-body">
                        Panel content
                    </div>
                </div>
            </div>
        </div>
        <div class="col-md-8">
            <div class="thumbnail">
                <!--预留的block盒子代码，格式为block...endblock-->
                {% block connect %}
                
                {% endblock %}
            </div>
        </div>
    </div>
</div>
</body>
</html>
```



在`order.html`文件中代码为空，不显示任何内容，但通过`extends`继承其他html文件后，展示继承内容

```html
<!--extends写在第一行，如果不写在第一行，那么block将找不到任何位置-->
<!--继承指定html文件中的所有内容-->
{% extends 'panel.html' %}

{% block title %}
<title>order</title>
<!--为block盒子结束符添加名字-->
{% endblock title %}

<!--block块是预留代码空间，与html文件中，名称为connect的block块对应。如果重写预留盒子中的内容，将改变原有内容；没有重写父盒子内容，则使用父盒子原有的内容，在此写的html代码将插入在html代码中的对应位置-->
{% block connect %}
<!--block.super表示父盒子中原有的内容-->
{{ block.super }}
<h3>这是order</h3>
{% endblock connect %}
```
