进程与线程的区别：

**1、进程的开销比线程大**

```python
import time
from threading import Thread

def task(name):
    print('%s is running'%name)
    time.sleep(2)
    print('%s is done'%name)

if __name__ == '__main__':
    t = Thread(target = task,args = ('子进程1',))
    t.start()

    print('主线程')
#线程程在执行的过程中，无需再创建内存空间，是使用的进程的内存空间，刚发起信号，就打印出了线程代码，所以开销比线程要小
'''
子进程1 is running
主线程
子进程1 is done
'''
```



**2、内存空间**

```python
import time
from threading import Thread

n = 100
def task(name):
    global n
    n = 0
    print('%s is running,n is %s'%(name,n))
    time.sleep(2)
    print('%s is done'%name)

if __name__ == '__main__':
    t = Thread(target = task,args = ('子进程1',))
    t.start()
    t.join()
    print('主线程',n)
#主线程和子进程是共用一块进程的内存空间，所以输出n的值是一样的
'''
子进程1 is running,n is 0
子进程1 is done
主线程 0
'''
```



3、pid

```python
import time,os
from threading import Thread,current_thread

def task(name):
    print('%s is running'%(name))
    time.sleep(2)
    print('在%s中，pid is %s'%(name,os.getpid()))
    print('在%s中，ppid is %s'%(name,os.getppid()))

if __name__ == '__main__':
    t = Thread(target = task,args = ('子线程1',))
    t.start()
    t.join()
    print('主线程',os.getpid())
#子线程和主线程的PID是一样的，因为它们都隶属于同一个进程
'''
子进程1 is running
在子线程1中，pid is 1124
在子线程1中，ppid is 511
主线程 1124
'''
```
